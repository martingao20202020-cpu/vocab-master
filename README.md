# VocabMaster 📚 智能单词记忆

科学记忆英语单词背诵卡片应用，基于艾宾浩斯记忆曲线的间隔重复算法。

## ✨ 功能特性

- 🔊 **单词发音** - Web Speech API 自动朗读
- 📌 **词根分析** - 拆解词根词缀，理解词源
- 📖 **双语释义** - 英文+中文双语定义
- 💡 **记忆方法** - 谐音/联想/故事等记忆技巧
- 📊 **记忆曲线** - SM-2 间隔重复算法科学管理复习
- ➕ **智能添加** - 输入单词自动查询/推荐关联词
- 📈 **学习统计** - 热力图/掌握度/连续天数追踪
- 💾 **数据管理** - 导入导出数据备份
- 📱 **移动适配** - PWA支持，可添加到手机主屏幕

## 🚀 部署到 GitHub Pages

1. 在 GitHub 上创建新仓库
2. 上传所有文件到仓库
3. Settings → Pages → Source → Deploy from branch (main)
4. 访问 `https://你的用户名.github.io/仓库名/`

## 📱 手机使用

在手机浏览器打开后，可"添加到主屏幕"作为 App 使用。

## 🧠 算法说明

使用 SM-2（SuperMemo 2）间隔重复算法：
- **再来** (质量0): 重置间隔，需要重新学习
- **困难** (质量3): 增加间隔，但减小记忆因子
- **良好** (质量4): 正常增加间隔
- **简单** (质量5): 大幅增加间隔，提高记忆因子
